## 常见问题

1. 明明是中文文档, 为什么不统一使用全角标点?
   > 因为文档站点使用的字体(Noto Sans)不支持 Chromium 排版引擎的标点挤压特性, 也没有找到其他支持该特性的适用于网页的字体.
